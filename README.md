# YukinekoPlayer

YukinekoPlayer is a high-end media player designed exclusively for macOS and Apple Silicon.
"Extended Linear Display P3" engine creates the ultimate viewing experience.

## Features
- Native Metal Rendering (16-bit Float Pipeline)
- High-Fidelity Audio (32-bit Float Engine)
- "Consord" Plugin Rack System
